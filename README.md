# Local-Search-Engine
ЛОКАЛЬНЫЙ ПОИСКОВЫЙ ДВИЖОК.
Search Engine (searchengine) – приложение, созданное на основе фреймворка Spring Boot с соблюдением концепции MVC. Оно способно индексировать сайты, ссылки на которые указаны в конфигурационном файле. Эта программа извлекает DOM-модель главной страницы, после чего с помощью технологии ForkJoinPool рекурсивно обходит ссылки на все страницы ресурса. 
Получаемый контент анализируется лемматизатором: из него извлекаются леммы слов, которые помещаются в отдельную таблицу. Связка леммы и страницы образует поисковый индекс – уникальное сочетание параметров, необходимое для быстрого поиска нужной информации.
Для работы приложения потребуется предварительно установить реляционную СУБД MySQL. Взаимоействие с программой происходит через графический интерфейс: достаточно запустить поисковый движок и подулючиться к нему через localhost:8080. В файле application.yaml не забудьте указать свои username и password для коннекта с базой данных.

ТЕХНОЛОГИИ
-Spring Boot
-Jpa-репозитории
-ForkJoinPool для поиска ссылок
-лемматизатор Apache Lucene
-MySQL
-концепция MVC

Как запустить проект? 
1. Клонируйте репозиторий с помощью команды git clone 
2. Установите зависимости (mvn install)
3. Запустите приложение
